Why Choose Rimmel?

Stream-oriented programming meets modern web development

No Virtual DOM

Direct DOM manipulation with optimized reactive sinks. Faster than Virtual DOM with zero unnecessary re-renders.

Stream-Oriented

Everything is a stream. Use RxJS observables for all state and behavior. Clean, testable, and maintainable code.

Lightweight

Minimal bundle size with tree-shakeable features. Optional build tools ‚Äî works great with plain JavaScript.

See It In Action

A simple reactive counter in just a few lines

JavaScript + Rimmel
import { rml } from 'rimmel';
import { BehaviorSubject, scan } from 'rxjs';

// Create a reactive stream
const clicks = new Subject();
const counter = clicks.pipe(
  scan(count => count + 1, 0)
);

// Bind to DOM with reactive template
document.body.innerHTML = rml`
  <button onclick="${clicks}">
    Clicked ${counter} times
  </button>
`;
